HOW TO USE THIS WEBSITE

1. Open index.html in your browser to preview.
2. Keep index.html, styles.css, and script.js in the same folder.
3. The Free Website Idea section is aligned without an embedded Google Form.
4. The "Fill Out the Form" button is still connected to your Google Form and opens it in a new tab.
5. Visitors can also contact you using the WhatsApp / Text button or the Call / Text button.
6. Upload index.html, styles.css, and script.js together to GitHub Pages, Netlify, Vercel, or your hosting provider.
7. Edit the text, packages, prices, form link, and phone number before sharing if needed.

This version includes:
- Light fall colors
- Fixed alignment
- Mobile responsive layout
- Google Form connected by button only
- No embedded Google Form or iframe
- Hamburger menu on mobile
- Floating leaves
- Scroll animations
- Animated counters
- Filterable website style cards
- 3D tilt effects on desktop
- Many large picture sections
